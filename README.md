# Clinical Readmission Intelligence Engine

End-to-end healthcare analytics platform designed to analyze hospital readmission risk and operational performance.

## Project Overview

This project demonstrates how modern data engineering and analytics tools can be used to build a clinical intelligence platform for hospital data analysis.

The platform integrates multiple data pipelines and analytical layers to identify high-risk patient cohorts and monitor healthcare performance metrics.

## Architecture

Data Sources  
↓  
Azure Data Factory (Ingestion)  
↓  
Databricks (Data Processing)  
↓  
PostgreSQL / Data Warehouse  
↓  
dbt Models  
↓  
Power BI Dashboards

## Key Features

- Healthcare readmission analytics
- End-to-end data pipeline architecture
- Data transformation workflows
- Interactive analytics dashboards
- Risk cohort identification

## Technology Stack

- Azure Data Factory
- Databricks
- PostgreSQL
- dbt
- Power BI
